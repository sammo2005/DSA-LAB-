# Arrays

## Description

This program demonstrates basic array operations in C.

### Operations Included

- Traversing an array
- Inserting an element
- Deleting an element
- Searching for an element

## Language

C Programming

## Complexity

| Operation | Time |
|-----------|------|
| Traverse | O(n) |
| Insert | O(n) |
| Delete | O(n) |
| Search | O(n) |
