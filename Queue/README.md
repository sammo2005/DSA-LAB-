# Queue

## Overview

This program implements a Queue (FIFO - First In First Out) using an array in C.

## Files

- `queue.c` - Queue implementation using an array.

## Operations

- Enqueue – Insert an element.
- Dequeue – Remove an element.
- Peek – View the front element.
- Display – Print all queue elements.

## Time Complexity

| Operation | Complexity |
|----------|------------|
| Enqueue | O(1) |
| Dequeue | O(1) |
| Peek | O(1) |
| Display | O(n) |
