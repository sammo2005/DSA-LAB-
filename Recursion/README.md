# Recursion

## Description

This folder contains basic recursive programs written in C. The programs demonstrate how a function can call itself to solve a problem.

## Programs Included

### 1. Fibonacci Series (Recursive)
- Generates the Fibonacci sequence using recursion.
- The user can specify the starting value and the number of terms to print.

### 2. Sum of Series (Recursive)
- Calculates the sum of the first **n** natural numbers using recursion.
- Example:
  - Input: 5
  - Output: 15

## Concepts Used

- Recursive Functions
- Base Case
- Recursive Calls
- Function Parameters

## Language

- C Programming

## Time Complexity

| Program | Time Complexity |
|---------|-----------------|
| Fibonacci (Recursive) | O(2ⁿ) |
| Sum of n Numbers | O(n) |

## Memory

The programs use the function call stack for recursion and do not require dynamic memory allocation.
