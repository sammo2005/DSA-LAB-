# Stack

## Overview

Implementation of a stack (LIFO - Last In First Out) data structure using an array in C.

## Files

* `stack.c` - Stack implementation using array

## Key Concepts

* **LIFO**: Last In First Out principle
* **push**: Insert element at the top
* **pop**: Remove element from the top
* **peek**: View top element without removing it
* **top**: Index pointing to the topmost element

## Operations Implemented

* Push
* Pop
* Peek
* Display

## Complexity

| Operation | Time |
| --------- | ---- |
| Push      | O(1) |
| Pop       | O(1) |
| Peek      | O(1) |
| Display   | O(n) |
